# Graveyard Gunner

Unreal Engine 5.6 project made during CG Spectrum [Game Programming Foundations](https://www.cgspectrum.com/courses/game-programming-foundations) course

See [this link for a video](https://www.youtube.com/channel/UC5SIMLDCKvBWfWDTwZjf_eg) of the game in action!

## Features

- Weapon system
- Material system
- Dismemberment
- Blood and gore FX
- AI-controlled enemies
- Camera shake
- Audio
- UI
- Save system
